About This Demo

This example demonstrates the vega-neutrino integration, which enables Vega to handle datasets 10-100x larger than normal by using columnar storage and SIMD-accelerated operations.

  • 3-5x memory reduction through columnar compression
  • 5-10x faster aggregations with SIMD acceleration
  • Background processing via Web Workers (for datasets >100K rows)
  • Transparent integration - same Vega specs work with minimal changes
