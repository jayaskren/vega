Instructions

1. Generate Test Data

Run the data generation script to create CSV and .ntro files:

cd examples && node generate-benchmark-data.js

2. Run Tests

Select a data size and click one of the test buttons. Each test opens in a new tab to ensure isolation.

3. View Results

Results are saved automatically and displayed in the table above. Run both Vega and Neutrino tests to see the comparison metrics.

4. Memory Measurements

For accurate memory measurements, start Chrome with:

chrome --enable-precise-memory-info
